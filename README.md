## Запуск проекта

1. Установите зависимости:

```bash
npm install
```

2. Запустите dev-сервер:

```bash
npm run dev
```

3. Откройте в браузере адрес, который покажет Vite (обычно `http://localhost:5173`).

## Страницы

- **/login** – страница логина:
  - Тексты: «Добро пожаловать», «Пожалуйста, авторизируйтесь».
  - Поля ввода логина и пароля.
  - Галочка «Запомнить данные»:
    - при включении сессия сохраняется в `localStorage`,
    - при выключении – в `sessionStorage`.
  - Кнопка «Войти» и подпись «Нет аккаунта? Создать».

- **/second** – защищённая страница (пока заглушка), доступна только после авторизации.
